# 🌍 Dream Stay – Full Stack Web Application

Dream Stay is a full-stack web application that allows users to explore, create, and review property listings.
The platform enables users to upload images, view property locations on an interactive map, and manage listings securely with authentication.

## ⚙️ Installation (Run Locally)

Clone the repository:

git clone https://github.com/Deep092004/Dream--Stay9.git

Navigate to the project folder:

cd Dream--Stay9

Install dependencies:

npm install

Create a `.env` file and add:

MONGO_URL=your_mongodb_connection_string
SECRET=your_session_secret

CLOUD_NAME=your_cloudinary_name
CLOUD_API_KEY=your_cloudinary_api_key
CLOUD_API_SECRET=your_cloudinary_api_secret

MAP_TOKEN=your_mapbox_token

Start the server:

node app.js

Open in browser:

http://localhost:8080
